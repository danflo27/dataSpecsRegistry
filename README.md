## Overview <a name="overview"> </a>  

<b>DataSpecsRegistry</b> is an on-chain registry for tellor oracle query types. It is a work in progress and is not yet live.

## Setting up and testing

Install Dependencies
```
npm i
```
Compile Smart Contracts
```
npx hardhat compile
```

Test Locally
```
npx hardhat test
```
